## 📋 About

I prepared this website as a quick demo for my neighbor in a single day. The project did not continue afterwards, but I am sharing it here as an archive so it doesn't get lost, as it might be useful in the future.

## ✨ Features <a id="features"></a>

- **Dynamic Product Catalog**: An extensive list where products can be filtered by category.
- **Advanced Search**: Real-time search functionality within product titles and descriptions.
- **Mobile Friendly**: A design that works seamlessly on all devices and a mobile-specific bottom menu.

## 🛠️ Technologies <a id="technologies"></a>

This project was developed using the following modern technologies:

- **React 19**: User interface components.
- **Vite**: High-performance development and build tool.
- **Vanilla CSS**: Custom-designed buttons and components.
- **React Icons**: Modern icons for the interface.
- **Kintaro UI**: My own component library.

## 🚀 Installation <a id="installation"></a>

Follow these steps to run the project on your local machine:

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/xkintaro/ovakids.git
   cd ovakids
   ```

2. **Install Dependencies:**

   ```bash
   npm install
   ```

3. **Start the Development Server:**
   ```bash
   npm run dev
   ```
   You can view the project by going to `http://localhost:5173` in your browser.
